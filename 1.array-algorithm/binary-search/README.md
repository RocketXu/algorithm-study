# 二分查找的原理以及实现
二分查找的实现原理非常简单，首先要有一个有序的列表。但是如果没有，则该怎么办？可以使用排序算法进行排序。以升序为例子：然后进行折半查找，找到中间值，然后目标值大于中间值就是在后半段；目标值小于中间值就是在前半段。
`在做题目时也非常容易识别：找就是对于找值得相关题目`
